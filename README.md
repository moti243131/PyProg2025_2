## Постановка задачи

Реализация классической игры «угадай число» с двумя методами поиска:

- **Линейный перебор** — медленный поиск слева направо
- **Бинарный поиск** — быстрый поиск в отсортированном списке

Алгоритм принимает число для поиска, список значений и метод угадывания. Возвращает найденное число и количество сравнений.

## guess_number.py

### Функция `guess_number`

Находит число в списке и возвращает его вместе с количеством сравнений.

**Сигнатура:**
```python
def guess_number(
    target: int,
    numbers_list: list[int],
    method: Literal["linear", "binary"],
) -> tuple[int, int]:
```

**Параметры:**
- `target` — искомое число
- `numbers_list` — список чисел (может быть не отсортирован, без дубликатов)
- `method` — `"linear"` (перебор) или `"binary"` (бинарный поиск)

**Возвращает:** кортеж `(угаданное_число, количество_сравнений)`

**Исключения:** `ValueError` — если `target` отсутствует в списке

**Формирование списка:**
```python
numbers = list(range(начало_диапазона, конец_диапазона + 1))
```

### Функция `input_range_from_keyboard`

Запрашивает начало и конец диапазона с клавиатуры.

**Возвращает:** кортеж `(start, end)`

**Поведение:** при некорректном вводе (не число, start > end) повторяет запрос.

## Алгоритмы

### Линейный поиск (linear)

- Проход по списку слева направо
- Сравнение каждого элемента с `target`
- **Сложность:** O(n) сравнений в худшем случае

### Бинарный поиск (binary)

- Список сортируется (создаётся `sorted(numbers_list)`)
- На каждой итерации: сравнение с серединой диапазона, сужение интервала
- **Сложность:** O(log n) сравнений

## Примеры использования

```python
from guess_number import guess_number, input_range_from_keyboard

# Список 1..100
numbers = list(range(1, 101))

# Линейный поиск
result = guess_number(42, numbers, "linear")
print(result)  # (42, 42) — 42 сравнения

# Бинарный поиск
result = guess_number(42, numbers, "binary")
print(result)  # (42, 7) — 7 сравнений

# Неотсортированный список (binary сортирует внутри)
numbers = [50, 10, 30, 20, 40]
result = guess_number(30, numbers, "binary")
print(result)  # (30, 2)
```

Ввод диапазона с клавиатуры:
```python
start, end = input_range_from_keyboard()
numbers = list(range(start, end + 1))
```
